## 🌟 Featured Projects

### 🎯 [ArticulateAI – Visual Speech Recognition](https://github.com/Ragita-kumari/ArticulateAI)
> A deep learning-based **lip reading system** that predicts spoken words from silent video clips — no audio needed!
- 🧠 Architecture: CNN Encoder → BiLSTM → Attention Mechanism → Classifier
- 🛠️ Built with: Python, PyTorch, OpenCV, Streamlit
- 📹 Upload any silent video and get real-time word predictions
- 💡 Useful for assistive tech for hearing-impaired users

### 📰 [Fake News Detection System](https://github.com/Ragita-kumari/fake_news_detection_system)
> NLP-based machine learning system to detect and classify fake news articles
- 🛠️ Built with: Python, NLP, Machine Learning

### 🔲 [QR Code Generator](https://github.com/Ragita-kumari/QR-Generator)

- 🛠️ Built with: HTML, CSS, JavaScript
